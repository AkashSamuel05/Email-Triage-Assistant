# Email-Triage-Assistant
Overview:
Email Triage Assistant is a smart email management tool that automatically sorts emails, recognizes priority, creates reply drafts, and declutters the inbox. The system communicates with Gmail/Outlook APIs and employs machine learning and natural language processing to optimize email handling.
This tool extracts the main points from email threads, shortens lengthy discussions, and offers automated decision, making help to cut down the time spent on email management.


Objectives:
.Automate email classification and prioritization
.Detect urgent emails
.Generate response drafts
.Extract meeting information
.Organize emails into smart folders
.Track follow-ups and productivity metrics


Core Features:
*Email Auto-Categorization
Classifies emails into:
.Important
.Work
.Personal
.Spam

*Priority Detection
Identifies urgent emails using keyword scoring
Assigns priority levels (High / Medium / Low)

*Thread Compression
Summarizes long email threads
Reduces processing overhead and improves efficiency

*Draft Response Generator
Provides template-based automated replies

*Meeting Extraction
Extracts date and time information from emails

*Follow-Up Tracker
Tracks emails requiring responses and reminders

*Smart Folders
Automatically organizes emails by category


System Architecture:-
1st: Email Provider (Gmail/Outlook)
2nd: Email Fetcher
3rd: Thread Compression
4th: AI Triage System
     _Classification
     _Priority Scoring
5th: Response Generator
Final: Smart Folders + Dashboard
